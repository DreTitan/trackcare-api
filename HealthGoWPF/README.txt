===============================================
HEALTHGO COLLECTION MANAGER - GUIA DE INSTALAÇÃO
===============================================

PASSO 1: INSTALAR O .NET 8 RUNTIME
--------------------------------------
Se ainda não tiver o .NET 8 instalado:
1. Acesse: https://dotnet.microsoft.com/download/dotnet/8.0
2. Baixe "Runtime Desktop .NET 8.0" para Windows
3. Execute o instalador e siga as instruções


PASSO 2: COPIAR A PASTA
--------------------------------------
1. Extraia o arquivo ZIP "HealthGoWPF" para qualquer local
2. Recomenda-se criar uma pasta "HealthGo" na Área de Trabalho


PASSO 3: EXECUTAR O PROGRAMA
--------------------------------------
1. Navegue até a pasta extraída
2. Procure o arquivo "HealthGoWPF.exe"
3. Clique duas vezes para abrir


PASSO 4: USAR O SISTEMA
--------------------------------------
O sistema já está configurado para conectar à nuvem automaticamente.
Todos os dados são sincronizados em tempo real entre todos os PCs
que estiverem usando o sistema.

Funcionalidades:
- Dashboard com métricas
- Cadastro de recolhimentos
- Anexar PDFs
- Comentários por setor
- Alteração de status
- Busca e filtros


RESOLUÇÃO DE PROBLEMAS
--------------------------------------

Erro ao abrir:
- Verifique se o .NET 8 Runtime está instalado
- Tente executar como Administrador

Não conecta na nuvem:
- Verifique sua conexão com a internet
- O serviço do Supabase pode estar em manutenção

Erro ao anexar PDF:
- Verifique se o arquivo é .pdf
- Verifique se há espaço em disco


CONTATO / SUPORTE
--------------------------------------
Em caso de dúvidas ou problemas, entre em contato com o
desenvolvedor do sistema.
